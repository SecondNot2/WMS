## 📦 Mapping URL Express → Next.js

Tất cả endpoints giữ nguyên path, chỉ thêm prefix `/api/`:

| Express                     | Next.js Route Handler           |
| --------------------------- | ------------------------------- |
| `POST /auth/login`          | `POST /api/auth/login`          |
| `GET /products`             | `GET /api/products`             |
| `POST /inbound/:id/approve` | `POST /api/inbound/:id/approve` |
| `GET /reports/export`       | `GET /api/reports/export`       |
| ... (62 endpoints)          | ... (giống path, prefix `/api/`)|

Frontend đã set `baseURL: '/api'` → call hiện tại như `/products` tự động → `/api/products`.

## 🚀 Deploy Vercel

1. Push repo lên Git
2. Vercel Dashboard → Add Project → Import repo (Root = `./`)
3. Add env vars (xem `apps/web/.env.production.example`)
4. Deploy → Vercel chạy: `pnpm install` → `prisma generate` (postinstall) → `prisma migrate deploy` → `next build`
5. Xong — có URL sống

Chi tiết: `docs/DEPLOY_VERCEL.md`
